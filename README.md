# Investment Evaluation Research

투자 판단 과정 평가 프레임워크 실험 프로젝트입니다.

## 목적

본 프로젝트는 사용자의 투자 판단 근거를 KURE-v1 요인 사전과 5개 평가항목을 기반으로 평가하는 규칙 기반 평가 엔진을 구현하고, 사람 평가 점수와의 일치도를 확인하기 위한 실험 코드입니다.

## 현재 범위

Phase 1에서는 LLM과 신경망을 사용하지 않습니다.

- 사람 라벨링 기반 요인 입력
- 규칙 기반 M1~M5 점수 계산
- 사람 평가 점수와 시스템 점수 비교
- 논문용 결과표 생성

## 향후 확장

- Phase 2: 신경망 기반 KURE-v1 요인 자동 라벨러
- Phase 3: LLM 기반 최종 피드백 생성
