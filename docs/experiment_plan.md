# Experiment Plan

## 1. 연구 목적

본 실험의 목적은 KURE-v1 요인 사전과 5개 평가항목을 활용한 규칙 기반 투자 판단 평가 프레임워크가 사람 평가와 어느 정도 일치하는지 확인하는 것이다.

## 2. 실험 대상 시나리오

- 시나리오 ID: S001
- 시나리오명: 금융 불안 심리 확산
- 투자 판단 대상: 국내 금융주 또는 금융주 중심 포트폴리오

## 3. 데이터 구성

| 구분 | 개수 | 설명 |
|---|---:|---|
| 우수 답변 | 10개 | 위험 요인과 완화 요인을 균형 있게 고려 |
| 보통 답변 | 10개 | 일부 핵심 요인만 반영 |
| 미흡 답변 | 10개 | 근거 부족, 감정적 판단, 과도한 행동 |

총 30개 답변을 사용한다.

## 4. 라벨링 방식

각 답변에 대해 2명의 평가자가 다음 항목을 라벨링한다.

- 언급 요인
- 누락한 필수 요인
- 진단 태그
- M1~M5 점수
- 평가 메모

## 5. 평가항목

| 코드 | 평가항목 |
|---|---|
| M1 | 핵심 정보 식별도 |
| M2 | 정보 해석 정확도 |
| M3 | 위험 인식도 |
| M4 | 행동-근거 정합성 |
| M5 | 논리 일관성 |

## 6. 시스템 평가 방식

사람이 라벨링한 `mentioned_factors`와 `diagnostic_tags`를 입력으로 사용하여 규칙 기반 엔진이 M1~M5 점수를 계산한다.

최종 점수는 다음과 같이 계산한다.

```text
Final Score = (M1 + M2 + M3 + M4 + M5) / 5
